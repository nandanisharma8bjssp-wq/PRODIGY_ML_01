# PRODIGY_ML_01: House Price Prediction Model

This repository contains **Task 01** completed during my **Machine Learning Internship at Prodigy InfoTech**.

---

## 📋 Task Overview

Implement a **Linear Regression** model to predict house prices based on three fundamental features:

- **Square Footage** (`GrLivArea`)
- **Bedrooms** (`BedroomAbvGr`)
- **Bathrooms** (`FullBath + HalfBath`)

---

## 📊 Dataset

The dataset used is the **House Prices – Advanced Regression Techniques** dataset from Kaggle.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📈 Performance Metrics

- **Mean Squared Error (MSE):** **2,806,426,667.25**
- **R-squared (R²) Score:** **0.6341** (~63.4% of variance explained)
- **Root Mean Squared Error (RMSE):** **52,975.72**
